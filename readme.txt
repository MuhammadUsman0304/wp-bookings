== Description ==

A simple and fast plugin to add booking funcationlites in your wordpress website, with fully customizeable UI

simply install the plugin activate it and from settings->wp chats you can add form links, live chat link, booking forms link. it will add a floating icon at the bottom of your website. 

add/change the size colors, icon from the settings page

== Installation ==

1. Upload wp chats to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
